１．まず始めに、以下（ａ〜ｅ）BBS用設定ファイル
　　"js/bbs_conf.js" を、必要に応じて書き換えて下さい。
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　※書換えは、文字コードを "UTF-8" で保存する必要がありますので
　　テキストエディタなどで、編集して下さい。
　　ジオシティーズのアドバンストHTMLエディタで編集すると
　　EUC に変換されてしまうようです。


　ａ．BBSユーザ名の設定（必須）
　~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　　１行目：USER_NAME = 'ebs_icon';

　　'ebs_icon' を、ご自分のホームページIDに変更してください。
　　※URLが http://xxx.geocities.jp/home_page_id/bbs/ なら、
　　　'home_page_id' が、ご自分のホームページIDになります。


　ｂ．タイトル欄 空投稿時の動作設定
　~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　　２行目：CHK_TITLE = 0;

　　タイトル欄が空の投稿をエラーにしたい場合は 1 を設定します。
　　初期値は 0:タイトル空白時 DEF_TITLE の文字列を設定します。


　ｃ．お名前欄 空投稿時の動作設定
　~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　　３行目：CHK_AUTH = 0;

　　お名前欄が空の投稿をエラーにしたい場合は 1 を設定します。
　　初期値は 0:お名前欄空白時 DEF_AUTH の文字列を設定します。


　ｄ．タイトル欄 空投稿時の文字列設定
　~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　　４行目：DEF_TITLE = 'タイトルなし';

　　タイトル欄が空で投稿された時に設定したい文字に変更して下さい。
　　CHK_TITLE = 0 の場合のみ有効になります。


　ｅ．お名前欄 空投稿時の文字列設定
　~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　　５行目：DEF_AUTH = 'ななしサマ';

　　お名前欄が空で投稿された時に設定したい文字に変更して下さい。
　　CHK_AUTH = 0 の場合のみ有効になります。


　※ｄ、ｅ で、文字列を設定したくない場合は '　'（全角スペース）で代用して下さい。
　　例：DEF_AUTH = '　';


２．"index.html" を必要に応じて編集して下さい。
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　５行目　：BBSのページタイトルです。
　42行目〜：BBSのタイトル〜メッセージです。
　119行目 ：メールアドレスの入力欄を有効にしたい場合は、
　　　　　　コメントアウトを解除（ <!-- と --> を削除）して下さい。


３．アップロードについての説明
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
　・BBSソース一式に含まれる全てのファイルをアップロードして下さい。
　　既存のBBSを残したい方は、BBSの設置ディレクトリを変えるか、
　　元のファイルを全てバックアップして置いて下さい。

　・既に利用されている掲示板に適用するにあたりまして、
　　３階層（返信への返信）よりも深いメッセージが存在する場合、正常に動作しません。
　　３階層より深いメッセージを削除する必要があります。


　・BBSソース一式（アップロードが必要なファイル一覧）

　　[BBSを設定したいディレクトリ]
　　　│
　　　├― index.html
　　　│
　　　├― js/
　　　│　 ├ bbs.js
　　　｜　 ├ bbs_conf.js
　　　｜　 └ treeview.js
　　　│
　　　└― css/
　　　　　 ├ bbs.css
　　　　　 └ screen.css

　　　※上記のファイルは全て 文字コード UTF-8 で設定しています。
